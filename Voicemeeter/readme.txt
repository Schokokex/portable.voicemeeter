Voicemeeter package includes VB-CABLE (Audio Device driver)
-----------------------------------------------------------------

Voicemeeter Virtual Mixing Console is Programmed by V.Burel
Copyrights V.Burel©2010-2020 All rights reserved.

Voicemeeter includes VB-CABLE technology to provide one virtual 
audio i/o. VB-CABLE is an Audio Driver working as virtual audio cable.

Voicemeeter package also installs: 
   - 1x ASIO Driver (32 and 64 bits)
   - Voicemeeter Remote API (32 and 64 bits)
   - Macro Button application

To support our effort and development team, END USER can make 
donation on www.VB-CABLE.com or www.VOICEMEETER.com web site. 
Thanks by advance for your participation !

For Commercial / professional use (in volume), thanks to 
contact us to get quotation and specific licensing terms.

INSTALLATION
-----------------------------------------------------------------

Just run the VoicemeeterSetup program (! IN ADMINISTRATOR MODE !).

REBOOT Must be done after installation (and after de-installation).
